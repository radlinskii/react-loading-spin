# react-loading-spin
A simple spinning loading animation react component.


## Preview
![Alt Text](https://imgur.com/UaME3w9.gif)

### Installing
```
npm i --save react-loading-spin
```

## Usage
```
...
import LoadingSpin from 'react-loading-spin';
...

...
render() {
  return (
    <div>
      <LoadingSpin/>
    </div>
  );
}
...
```
